## ⚠️ 既知の制約・課題

### インポートパス調整
- **状況**: Python相対インポートの調整が必要
- **影響**: 直接スクリプト実行時のみ
- **対策**: パッケージとしての実行推奨

### インメモリ実装の制約
- **データ永続性**: アプリ再起動時に消失
- **スケーラビリティ**: 単一プロセス内のみ
- **対策**: 実DB実装への置き換え容易
